clplot

The following is documentation on the command line chart generator cplot. For examples and the full project details, see the project post.

Description

An ascii chart generator for the terminal written in C++.

Installation

Git install:

git clone https://github.com/k-vernooy/clplot
make
make install
clplot -d [1,4,9,16,25] -c red
